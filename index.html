Política de Privacidade

Esta Política de Privacidade descreve o compromisso da PowerTech Industrial com a proteção de dados e a transparência. O aplicativo PowerTech é uma ferramenta de cálculos técnicos e gestão comercial offline.

⚠ NOTA IMPORTANTE: Este aplicativo opera de forma 100% OFFLINE. Nenhum dado inserido é transmitido, coletado ou armazenado em servidores de nuvem.

1. Coleta de Dados para Salvamento Interno

Para o funcionamento das ferramentas de gestão (Orçamentos, Clientes e Financeiro), o aplicativo permite que o usuário insira:

  • Dados do Profissional: Nome, Logotipo, CPF ou CNPJ e informações de contato para personalização de recibos.
  • Dados de Clientes: Nome completo, Endereço residencial ou comercial, CPF e CNPJ.
  • Gestão Comercial: Valores de serviços, detalhes de orçamentos e registros financeiros de entrada/saída.

2. Armazenamento Local e Exclusivo

Todos os dados inseridos são armazenados exclusivamente no banco de dados local do dispositivo móvel do usuário (utilizando armazenamento interno seguro do sistema Android). A PowerTech Industrial não possui acesso a esses dados.

3. Compartilhamento de Dados por Terceiros

O aplicativo não compartilha dados com terceiros. A única forma de exportação de dados ocorre por ação direta e deliberada do usuário, como ao gerar e enviar um orçamento em PDF via e-mail ou aplicativos de mensagens.

4. Permissões de Arquivos

O aplicativo solicita permissão para acessar o armazenamento para fins de salvamento e leitura de documentos gerados (PDF/Imagens), garantindo que os orçamentos e recibos emitidos fiquem disponíveis para o usuário.

5. Exclusão de Dados

Como os dados ficam apenas no celular, a exclusão definitiva das informações ocorre automaticamente quando o usuário opta por desinstalar o aplicativo ou limpar os dados do app nas configurações do Android.
