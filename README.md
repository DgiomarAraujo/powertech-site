# PowerTech Industrial - Engenharia e Soluções Tecnológicas

Este repositório contém o código-fonte do site oficial da **PowerTech Industrial**, com foco em soluções de engenharia elétrica, energia solar e ferramentas técnicas para o setor industrial em Floriano, Piauí.

## 🚀 Sobre o Projeto
O site serve como vitrine profissional e central de ferramentas, oferecendo:
*   **Apresentação de Serviços:** Engenharia elétrica, montagem de painéis industriais e projetos de energia solar.
*   **Calculadoras Técnicas:** Ferramentas web para cálculo de queda de tensão, dimensionamento de condutores e geração fotovoltaica.
*   **Acesso ao Aplicativo:** Link para download da ferramenta oficial PowerTech para Android.

## 🛠️ Tecnologias Utilizadas
*   **HTML5 & CSS3:** Estrutura e estilização responsiva.
*   **JavaScript:** Lógica de cálculo das ferramentas técnicas.
*   **GitHub Pages:** Hospedagem e deploy automatizado.

## 📱 Aplicativo PowerTech (Android)
O projeto também se integra ao ecossistema do aplicativo desenvolvido em **Java (Android Studio)**, garantindo que as políticas de privacidade e suporte técnico estejam centralizadas neste domínio.
